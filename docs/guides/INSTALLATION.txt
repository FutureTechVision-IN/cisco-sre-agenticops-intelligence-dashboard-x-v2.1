QUICK START (2 minutes):
1. chmod +x start.sh stop.sh
2. ./start.sh
3. Visit http://localhost:8001
4. Login: sre-admin / password$$

SYSTEM REQUIREMENTS:
- Node.js 18+ (check: node -v)
- npm 9+ (check: npm -v)
- PostgreSQL 15+ database
- 100 MB+ free disk space

✓ Cisco CIRCUIT AI-powered analytics engine
✓ PDF & CSV report generation
✓ User authentication & role-based access
✓ Automated startup/shutdown scripts

- Automatically installs npm dependencies

- Seeds 98,966+ vulnerability records

- Starts application on port 8001

SHUTDOWN SCRIPT FEATURES:

- Process cleanup

- SESSION_SECRET=your-secure-session-secret
- PORT=8001

TROUBLESHOOTING:
- Port 8001 in use? Change port: PORT=3000 ./start.sh
- Database error? Set: export DATABASE_URL="postgresql://..."
- Node not found? Install from nodejs.org
- Memory error? Increase: NODE_OPTIONS="--max-old-space-size=4096"
