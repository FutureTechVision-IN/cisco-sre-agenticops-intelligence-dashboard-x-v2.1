# SRE Voice AI - Simple Design

## Quick Implementation

This is a simple, eye-catching icon combining:
- **Brain** (Cyan) - Represents intelligence/AI
- **Lightning Bolt** (Purple) - Represents power/striking energy
- **Glow effect** - Yellow accent for striking appearance
- **Pulsing animations** - Shows activity

## Usage

### React Component
```tsx
<img src="/assets/icons/sre-voice-ai-simple.svg" alt="SRE Voice AI" className="w-8 h-8" />
```

### HTML
```html
<img src="/icons/sre-voice-ai-simple.svg" alt="SRE Voice AI" width="56" height="56">
```

### CSS Background
```css
.sre-button {
  background-image: url('/icons/sre-voice-ai-simple.svg');
  background-size: contain;
  width: 56px;
  height: 56px;
}
```

## Colors Used
- **Cyan**: #06b6d4 (Brain)
- **Purple**: #a855f7 (Lightning)
- **Yellow/Amber**: #fbbf24 (Glow/Power indicator)

## Design Features
✅ Simple & clean (no complex gradients)
✅ Eye-catching (lightning bolt is striking)
✅ Animated (pulsing glow shows activity)
✅ 256×256 viewBox (scales infinitely)
✅ Minimal file size (~2KB)
